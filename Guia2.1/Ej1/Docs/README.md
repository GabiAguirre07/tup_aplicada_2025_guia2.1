# Ejercicio 1 – Eliminación en cascada con tabla relacional

Este ejercicio crea una base de datos con las tablas `Cursos`, `Alumnos` y `Cursos_Alumnos`.  
Se prueba la eliminación en cascada del curso **Matemática**, verificando que se borren las inscripciones en la tabla relacional sin eliminar los alumnos.

## Archivos
- `1_creando_base_de_prueba.sql`: Crea la base de datos y carga datos de prueba.
- `2_eliminar_matematica.sql`: Elimina el curso Matemática y muestra el efecto en las inscripciones.

## Modelo
- `Modelo_de_datos.jpg`: Diagrama entidad-relación.
- `Modelo_de_datos.uml`: Modelo UML con relaciones y restricciones.
