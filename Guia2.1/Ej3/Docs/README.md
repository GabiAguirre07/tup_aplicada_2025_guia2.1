# Ejercicio 3 – Consulta de alumnos y localidades

Este ejercicio crea una base de datos con las tablas `Alumnos` y `Localidades`.  
Se realiza una consulta con **LEFT JOIN** para mostrar todos los alumnos junto con su dirección y localidad.  
El objetivo es validar la integridad referencial y mostrar datos completos de los alumnos.

## Archivos
- `1_creando_base_de_prueba.sql`: Crea la base de datos y carga datos de prueba.
- `2_consulta_alumnos.sql`: Consulta alumnos con sus datos y localidades usando LEFT JOIN.
